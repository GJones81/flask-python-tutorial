Python Loops

For Loops

The 'For' loop itirates through a list, one item at a time, and assigns a variable to each loop. The variable allows you to write code that performs some action on each item in the list.

Here's a sample list:

basket = ['apple', 'orange', 'banana', 'strawberry', 'cantaloupe']

for fruit in basket:

fruit.upper()

return basket

The code above takes each item in 'basket', one at a time and names it 'fruit'. Then the upper() function changes each item to all uppercase letters

While Loops

The 'While' loop itirates through a list as long as a test case remains true. It is very important to ensure that a while loop will eventually hit the test case and stop. The While loop will continue until its test case is made false, or it is stopped by a stack-overflow error.

Here's an example

number = 0

while(number < 10):

number += 1

print(number)

The code above looks at the number variable, which begins at 0, and evaluates against the test case: number < 10. if number is less than 10, the code in the while loop is run. The value of number is increased by one, and printed. The cycle begins again at the test case, and repeats until the test case is made false.

Range Loops

The range() function can be used to loop through an object within a specified range, at a specified step size. The function takes up to three arguments: range(start, stop, step). It's important to remember that the starting argument is inclusive, but the stoping argument is exclusive(the loop will end BEFORE it get to the index specified). The stop argument is required

Sample code

x = range(3, 6)

for n in x:

print(n)

The code above will start at three and print 3, then move to 4 and print 4, and then move to 5 and print 5. The stopping argument is exclusive, so the function will stop before 6
